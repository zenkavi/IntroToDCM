Make input task

% Input - block task to node 1
num_blocks = 4;
on_len = 10;
off_len = 10;
stim = [zeros(1, off_len) ones(1, on_len) zeros(1, off_len)];
my_task_secs = repmat(stim, 1, num_blocks);
T = length(my_task_secs);
plot(0:T-1, my_task_secs)
axis([0 120 0 1.1])
Create new DCM structure with fields needed by tapas_rdcm_generate.m
Specify the input.
my_DCM.U.name = {'u1'};
my_DCM.U.dt = 0.5/16;
my_DCM.U.u = interp1(1:T, my_task_secs, 1:my_DCM.U.dt:T, "linear")'

Specify constants that'll be used for the balloon model.

A priori connectivity matrix (set to full connectivity to see if it can correctly prune it to the true matrix)

Specify the options for the rDCM analysis
options.SNR = 3;

Generate BOLD timeseries using created DCM structure
